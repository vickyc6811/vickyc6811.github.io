CYOA Project

My teacher assigned this project so we can express our knowlege using Github with creativity. This project helped me practice command lines, git, and merging when collaborating with groupmates.

Something that was difficult about this project was dealing with merge conflict when working on our project at the same time. Each groupmate had to work on different folders and files. It was frustrating because everyone was working in the ide at the same time and all trying to fix it at once. I overcame this challenge by communicating with our group and checking to see if they made any changes and all agreeing on what changes we want, so I can pull their changes and push my changes as well.

A thing I would've done differently if I had more time was including images, so the audience can be more enganged in our project. Visuals would have made our project look better and more appealing.
